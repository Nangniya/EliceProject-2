### 기획의도
- 코딩테스트를 풀때 제한시간을 안 두니 쉽게 늘어져버린다.
- 구글링이나 커뮤니티를보면 약간의 도움만 얻어보려 검색했는데 답지를 목격해버리는 일이 왕왕있다.
<br />
코딩테스트 사이트에 있는 문제를 가져와서 문제별로 타이머를 실행할 수 있고, 일정시간이 지나면 힌트를 순서대로 열람할 수 있다.
